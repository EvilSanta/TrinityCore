TrinityCore by TrinityCore

The Power of community

Trinity is an MMORPG framework that's built using C++, and using a back-end of MySQL for the DB content. Trinity was at one point a merger of MaNGOS with many patches submitted by users; That is no longer the case. Trinity is an individual project with a diverse development team, that also accepts code from our users. Trinity has been cleaned up a lot, and will continue to be cleaned up and optimized over time. Please help us by posting issues, patches, and enhancement suggestions on the tracker.

Dependencies

Some examples of dependencies : autoconf libtool gcc g++ make cmake git-core mercurial patch wget links zip unzip unrar openssl libssl-dev mysql-server mysql-client libmysqlclient15-dev libmysql++-dev libreadline5-dev zlib1g-dev libbz2-dev please check Check http://www.trinitycore.info for the latest info on dependencies.
